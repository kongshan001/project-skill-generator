```
┌─────────────────────────────────────────────────────────────┐
│                    每30分钟自动执行                           │
└─────────────────────────────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  1. 克隆/更新目标仓库                                         │
│     git clone https://github.com/kongshan001/{repo}.git     │
└─────────────────────────────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  2. 运行代码库分析                                           │
│     python scripts/analyze_codebase.py {repo}               │

                             │

│  3. 生成项目技能                                             │

                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  4. 生成专家代理                                             │
│     python scripts/generate_agent.py analysis.json          │
│     --output {repo}/.claude/agents/ --team                  │
└─────────────────────────────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  5. 启动 Claude Code 验证                                    │
│     - 加载生成的技能库                                        │
│     - 执行测试任务                                            │
│     - 记录结果                                                │
└─────────────────────────────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  6. 记录验证结果                                             │
│     - 成功/失败指标                                           │

│     - 改进建议                                                │
│     - 写入 doc/practices/{repo}_{date}.md                    │
└─────────────────────────────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  7. 更新 project-skill-generator                             │
│     - 修复发现的问题                                          │

│     - 更新 CHANGELOG.md                                       │
└─────────────────────────────────────────────────────────────┘
```

## 验证任务模板

每个项目验证时，使用以下测试任务：

### 通用测试任务
1. **理解项目架构**: "请根据技能库，总结这个项目的架构和核心模块"
2. **定位代码**: "找到处理用户认证的代码，并解释其工作原理"
3. **添加功能**: "在 {模块} 中添加一个新功能：{具体需求}"
4. **修复 Bug**: "修复 {具体问题}，确保不破坏现有功能"
5. **重构代码**: "优化 {代码片段}，提高可读性和性能"

### 项目特定测试任务
根据每个项目的特点，设计特定的验证任务。

## 自动化配置

### Cron Job 配置

```bash
# 每30分钟执行一次验证
*/30 * * * * /path/to/validate_next_repo.sh >> /var/log/psg-validation.log 2>&1
```

### 环境变量

```bash
export PSG_SKILL_DIR="/root/.openclaw/workspace-opengl/skills/project-skill-generator"
export PSG_REPOS_DIR="/root/.openclaw/workspace-opengl/repos"
export PSG_DOC_DIR="$PSG_SKILL_DIR/doc"
```
