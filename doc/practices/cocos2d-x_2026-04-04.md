### 需要更新的配置

在脚本配置部分添加大仓库检测：

```bash
# 大仓库列表（需要特殊处理）
LARGE_REPOS=(
    "cocos2d-x"
    "cpython"
    # 其他可能的大仓库
)

# 检查是否为大仓库
is_large_repo() {
    local repo=$1
    for large_repo in "${LARGE_REPOS[@]}"; do
        if [[ "$repo" == "$large_repo" ]]; then
            return 0
        fi
    done
    return 1
}
```
