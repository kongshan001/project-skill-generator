## 🚀 Usage

### Quick Start

```bash
# Validate installation
python3 validate.py

# Run demo
python3 demo.py

# Analyze codebase
python3 scripts/analyze_codebase.py /path/to/codebase

# Generate skills
python3 scripts/generate_skill.py analysis.json

# Generate agents
python3 scripts/generate_agent.py analysis.json --team
```

### Claude Code Integration

```bash
# Generate everything in one command
/generate-project-skills /path/to/your/codebase

# Result: .claude/ directory with skills/ and agents/
```
